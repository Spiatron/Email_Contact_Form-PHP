------------------------------------- Note: These are the important steps to follow ------------------------------------- 


Step 1:
  * Unzip and copy the EmailContactForm folder and paste into htdocs folder path C:\xampp\htdocs

Step 2:
  * Copy the file (php.ini) which is in configurationFiles Folder and paste it to the Following path: C:\xampp\php

Step 3:
  * Now, copy the file (sendmail.ini) which is in configurationFiles Folder and paste it to the Following path: C:\xampp\sendmail

Step 4:
  * Start the Apache server from Xampp control panel, if it is already started then restart it (IMPORTANT)

Step 5:
  * Open any browser and paste this link http://localhost/EmailContactForm/src/ContactForm.php
